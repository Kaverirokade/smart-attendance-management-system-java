### 🚀 AI-Based Face Recognition Attendance Management System

An intelligent attendance management system built using **Java**, **MySQL**, and **Python-based Face Recognition** to automate attendance tracking, user management, and reporting with a modern GUI interface.

## 📌 Features

### 👨‍💼 Admin Module
- Admin Login Authentication
- Register New Students
- Update Student Information
- Delete Student Records
- View Student Details
- Mark Attendance Automatically
- Attendance Analytics Dashboard
- Notifications Management

### 👨‍🎓 Student Module
- Student Login
- View Personal Details
- Update Profile Information
- Check Attendance Records
- Export Reports to PDF
- Export Reports to Excel

### 🤖 AI Face Recognition
- Face Registration using Camera
- Face Embedding Generation
- Real-Time Face Detection
- Automatic Attendance Marking
- Python Server Integration

### 📊 Data Visualization
- Pie Charts
- Line Charts
- Attendance Statistics Dashboard

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/your-username/smart-attendance-system-using-java.git
```

### 2. Open Project

Open the project in **NetBeans IDE**.

### 3. Configure Database

- Create MySQL Database
- Import SQL File
- Update Database Credentials

### 4. Start Python Face Recognition Server

```bash
python app.py
```

### 5. Run Java Application

Execute:

```bash
AppLauncher.java
```
